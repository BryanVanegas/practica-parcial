# 📘 Práctica de Parcial: **Git & GitHub**

> 💡 Este proyecto corresponde a una práctica realizada para el **parcial de Git y GitHub**.  
> Aquí aprenderás los **comandos básicos**, el **flujo de trabajo** y cómo manejar un repositorio remoto en GitHub.  

---

## 🎯 Objetivos
✔️ Comprender los conceptos de **control de versiones**.  
✔️ Aprender a manejar un repositorio en **GitHub**.  
✔️ Practicar comandos como `clone`, `commit`, `push`, `pull`, entre otros.  
✔️ Simular un **flujo de trabajo colaborativo**.  

---

## 🛠️ Tecnologías utilizadas
- 🐙 **Git** → Sistema de control de versiones distribuido  
- 💻 **GitHub** → Plataforma de alojamiento de repositorios remotos  
- 📄 **Markdown** → Documentación del proyecto  

---
